# Show Ideographic Space Package
This package show ideographic spaces ("　", U+3000, 和字間隔, known as FULLWIDTH SPACE).

# 和字間隔表示パッケージ
和字間隔("　"、U+3000、IDEOGRAPHIC SPAC、いわゆる全角スペース)を目に見える形で表示します。
